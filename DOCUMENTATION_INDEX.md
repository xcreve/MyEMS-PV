## 📌 重要提醒

### ⚠️ 在开发前必读:
- DEVELOPER_GUIDE.md 第 8-10 章 (CSS 和 API 规范)
- README.md 的 API 端点文档
- FEASIBILITY_PLAN.md Phase 1 的"路由统一"和"状态管理"

### ⚠️ 在修改主题时必读:
- DEVELOPER_GUIDE.md 的"CSS 主题系统"章节
- 记住规范: ✅ 使用 CSS 变量类名, ❌ 不要硬编码颜色

### ⚠️ 在提交代码前检查:
- 是否遵循了 DEVELOPER_GUIDE 的命名约定?
- 是否添加了 TypeScript 类型?
- 是否使用了 CSS 变量而不是硬编码颜色?
- 是否通过了 linting 和类型检查?
